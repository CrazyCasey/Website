User Stories

Guest

  • As a guest, if interested, I wish to find out more information about the entrepreneurial center.
  • As a guest, if interested, I wish to register for an account.
  • As a guest, I need to view companies requesting resources.
  • As a guest, I need to view links to EC social media web sites.
  • As a guest, I need to view upcoming EC events on the calendar.

Registered Users (Including Pending Approval)

  • As a registered user, I need to be notified of upcoming EC events.
  • As a registered user, I need to update my user profile, including my available skillset.
  • As a registered user, I need to be able to create alerts for resource requests meeting a given criteria.

Approved Users (@ODU.edu)

  • As an approved user, I need the same access as a Registered User.
  • As an approved user, I need to request a meeting with an EC mentor.
  • As an approved user, I need to request a new company.
  • As an approved user, I need to let others know that I can be a resource.
  • As an approved user, I need to be able to request a meeting with an EC mentor.

Student Company Spokesperson

  • As a student company spokesperson, I need to update both public and private information about the company.
  • As a student company spokesperson, I need to have the option of displaying information about the company.
  • As a student company spokesperson, I need to view and request available resources.
  • As a student company spokesperson, I need to view and approve requests from resources.
  • As a student company spokesperson, I need to collaborate with resources through chat and file sharing.
  • As a student company spokesperson, I need to schedule an available meeting room.

External Company Spokesperson

  • As an external company spokesperson, I need to update information about the company.
  • As an external company spokesperson, I need to be able add/edit company events to the calendar.
  • As an external company spokesperson, I need to post available resources of my company.
  • As an external company spokesperson, I need to approve requests to to join student companies.

Student Resources

  • As a student resource, I need to view companies requesting resources.
  • As a student resource, I need to send a request to be added to a company.
  • As a student resource, I need to approve requests to be added to a company.

Entrepreneurial Staff

  • As an EC staff member, I need to send out notifications to users.
  • As an EC staff member, I need to add/edit EC events to the calendars.
  • As an EC staff member, I need to view and approve non-ODU user accounts.
  • As an EC staff member, I need to view and approve requested companies.
  • As an EC staff member, I need to approve external mentors.
  • As an EC staff member, I need to assign/suggest external mentors and/or resources to companies.
  • As an EC staff member, I need to add/update/delete key words for resource searching.
  • As an EC staff member, I need to approve requests for meetings with an EC mentor.

Administrator

  • As an administrator, I need to be able to send out system notifications to users.
  • As an administrator, I need to be able to add and delete user accounts.
  • As an administrator, I need to be able to enable and disable user accounts.
  • As an administrator, I need to be able to add, edit, and archive companies.
  • As an administrator, I need to be able to configure white-listed account requirements.

Test User

  • As the test user, I need to be able to change the calendar date.
  • As the test user, I need to be able to change my view permissions.
  • As the test user, I need to be able to change my access permissions.

External Resource

  • As an external resource, I need to list what I have available to student ventures.
  • As an external resource, I need to be able to decline or approve the use of resources to student resources.
  • As an external resource, I need to state whether or not I will be available as a mentor.
  • As an external resource, I need to be able to decline or approve mentoring a student venture.
